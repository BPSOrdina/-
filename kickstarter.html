The Ordina BPS Kickstarter Programme focuses on young graduates that want to give their career a boost with our extensive two-month training.

Curriculum - What you'll learn

Identities

You´ll see things like: AD, Azure AD, Office 365,… The exercises cover GUI as well as PowerShell. You´ll see how to manage identities fast and efficiently. This training is a good start for acquiring server and cloud certificates later in your carrier.

Infrastructure

Server, networking & storage

This workshop will give you insight into how server, networking and storage equipment can seamlessly work together. This includes on-premise as well as cloud based solutions. The examples provided are based on real-life situations.

Infrastructure

Linux

If you´ve ever feared the command-line, this workshop will help you overcome it! Through hands-on-labs you will learn how to install, configure and maintain CentOS Server and services such as LAMP, Postfix, FTP, SSH,….By the end, you will be well on your way to becoming a true Linux ninja.

Development

Client side Development

Gain a basic knowledge on building modern client side applications targetting the Office 365 cloud platform.

Development

Powershell

When working with Microsoft products PowerShell is essential. The trainer will provide you with tips & tricks on how to start scripting in PowerShell. Along with exercises tailored to your existing knowledge. For example, making Hyper-V machines and managing them, configuring NTFS rights, …. You´ll see the most popular commands. Additional you´ll get large example scripts that give a good basis for making excellent scripts.

Development

Cloud development

Learn the basics about developing Azure cloud solutions. Topics include Azure storage, servicebus, queues,…

Development

Microsoft Exchange

In this workshop you´ll learn how to properly configure and manage Microsoft Exchange. Even though many enterprises are embracing Office 365, Microsoft Exchange still has its place in the market for handling mail traffic. You´ll also learn how a Microsoft Exchange server is used in a hybrid environment.

Communication

Agile/SCRUM

Have you ever wondered if an assignment could have been done better? It probably could have been. Taught by an external instructor, the SCRUM/Agile workshop sets the stage towards a more efficient way of working as an individual and as part of a team.

Process Management

Mobile Device Management

Nowadays people like to bring their own devices into the workplace. To make this viable, IT operations should reconsider the way devices are managed. Enter Mobile Device Management. This class will give you a high-level overview of the available tools that make it possible to manage smartphones, tablets, and traditional computers, whether they belong to the organization or to the individual.

Collaboration

Microsoft SharePoint

Learn the architectural insights & functional capabilities of Microsoft SharePoint and extend it with lightweight customizations

Collaboration

Soft Skills

In the soft skills training you´ll learn techniques to efficiently communicate in difficult situations. For example, hard job interviews, when you need documents to be able to proceed with your work, communicating with someone who doesn´t want to corporate. You´ll practice this skills with a communication expert and a professional actress for the simulations. Kickstarter alumni´s describe the simulations as hard but extremely useful.
